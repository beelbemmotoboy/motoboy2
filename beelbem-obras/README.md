# Beelbem Obras

Aplicativo independente para gestao visual de obras residenciais.

A versao atual tem navegacao completa, dados locais persistidos no navegador, cadastro de obra, busca/filtros, etapas, fotos, pendencias, PLS Caixa, insumos, ferramentas, checklist, normas e relatorios simulados.

Ainda nao ha autenticacao real, banco de dados externo ou analise de IA integrada.

## Rodar localmente

```bash
npm install
npm run dev
```

Build de producao:

```bash
npm run build
```
